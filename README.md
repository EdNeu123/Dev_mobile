## Como executar qualquer projeto

```bash
# Clone o repositório
git clone https://github.com/EdNeu123/nome-do-repositorio.git

# Entre na pasta do projeto desejado
cd nome_do_projeto

# Instale as dependências
flutter pub get

# Execute o app
flutter run
```

## Conceitos aplicados por aula

- **Aula 02 — Dart:** Classes, construtores, tipos, coleções
- **Aula 03 — Dart Async:** Future, async/await, Future.delayed
- **Aula 05 — Widgets:** StatelessWidget, Text, Container, Column, Row, Scaffold, AppBar
- **Aula 06 — Layouts:** MediaQuery, Expanded, Flexible, Wrap, Stack, Positioned, breakpoints
- **Aula 07 — Estado:** StatefulWidget, setState(), initState(), dispose(), TextField, Checkbox
- **Aula 08 — Navegação:** Navigator.push(), Navigator.pop(), passagem de dados entre telas
